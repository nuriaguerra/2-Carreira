# Deseño de Software

> Programación · Segundo curso — 2º cuatrimestre

## De que vai esta materia

En Deseño de Software realízase modelado con UML: diagramas de clases, secuencia, casos de uso... Aínda que pareza que non, o traballo da continua é complexo e non existen unhas directrices que seguir, polo que complícase bastante máis. O exame non ten nada que ver cos exercicios que se realizan ao longo do cuatrimestre, así que non entredes en pánico.
