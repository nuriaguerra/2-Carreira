# Sistemas Operativos II

> Sistemas · Segundo curso — 2º cuatrimestre

---

## De que vai esta materia

Sistemas Operativos II aprofunda nos temas máis avanzados que quedaron pendentes en SO I, e amplía o foco cara aos sistemas distribuídos e a administración real de sistemas. É onde a teoría de SO se converte en ferramentas que se usan no día a día profesional.

---

## Contidos habituais

- Xestión avanzada de memoria: asignación dinámica, fragmentación, GC
- Sistema de ficheiros en profundidade: implementación, xornalización, RAID
- E/S avanzada: E/S asíncrona, polling vs interrupciones, drivers
- Seguridade do sistema operativo: control de acceso, capacidades, SELinux
- Virtualización: hipervisores tipo 1 e tipo 2, contenedores (Docker)
- Sistemas distribuídos: RPC, sincronización distribuída, consistencia
- Introdución a Linux: administración, shell scripting, servizos do sistema
- Rendemento e monitorización: ferramentas de profiling e análise

## Consellos

- Aprende a usar o terminal de Linux con soltura — é imprescindible e esta materia é o mellor momento para facelo.
- Docker cambiou a forma de despegar software. Se o teu temario o inclúe, préstalle atención especial.
- O shell scripting parece menor pero aforra horas de traballo repetitivo. Vale a pena aprenderlo ben.
- A virtualización e os contenedores son tecnoloxías clave na industria actual — entendelas a fondo ten valor máis aló da materia.
