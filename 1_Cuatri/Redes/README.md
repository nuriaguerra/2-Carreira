# Redes

> Sistemas · Segundo curso — 1º cuatrimestre

---

## De que vai esta materia

Redes explica como os ordenadores se comunican entre si — desde o cable físico ata a aplicación que usas no navegador. É a materia que che permite entender o que pasa realmente cando abres unha páxina web, envías un correo ou fas unha videollamada.

Ten moita teoría de protocolos, pero ten aplicación directa e visible en todo o que fas co ordenador.

---

## Contidos habituais

- Modelo OSI e modelo TCP/IP: capas e funcións
- Capa física: medios de transmisión, codificación, ancho de banda
- Capa de enlace: Ethernet, MAC, switches, detección de erros
- Capa de rede: IPv4, IPv6, enrutamento, máscaras de subrede, CIDR
- Capa de transporte: TCP vs UDP, control de fluxo e conxestión
- Capa de aplicación: HTTP, DNS, DHCP, FTP, SMTP
- Seguridade básica en redes: firewalls, VPN, cifrado
- Redes sen fíos: WiFi, estándares 802.11
- Introdución ao enrutamento dinámico: RIP, OSPF

## Consellos

- Aprende a calcular subredes de cabeza: é exercicio frecuente en exame e non custa tanto como parece.
- O modelo OSI é un marco conceptual, non a realidade. TCP/IP é o que realmente se usa. Enténdeos os dous pero non os confundas.
- Wireshark é a ferramenta máis útil para esta materia — capturar tráfico real fai que a teoría encaixe moito mellor.
- TCP é complexo por deseño: foi construído para ser fiable nunha rede non fiable. Cando entendes ese obxectivo, os mecanismos teñen sentido.
